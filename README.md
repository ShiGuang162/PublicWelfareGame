## 📖 项目简介

《急救先锋》是一款以应急急救知识为核心的公益问答闯关游戏。玩家通过回答心肺复苏、止血包扎、火灾逃生等急救主题的题目，收集小红花、提升段位，在竞技闯关中潜移默化地掌握救命技能。

**一句话定位**：让每个人都能在游戏中学会急救，在关键时刻挺身而出。

### 评分机制

```
关卡得分 = (答对题数 × 20) + (剩余秒数 × 2) + (最高连击 × 5)
```

| 星级 | 条件 |
|------|------|
| ⭐⭐⭐ | 全部答对且剩余时间 > 30 秒 |
| ⭐⭐ | 全部答对或正确率 ≥ 80% |
| ⭐ | 正确率 ≥ 60% |
| 未通过 | 正确率 < 60% 或时间耗尽 |

## 🎮 游戏操作流程

1. **首页** — 输入昵称 → 查看当前段位和小红花 → 点击「开始闯关」
2. **选关** — 选择已解锁的关卡卡片 → 点击进入
3. **答题** — 每关 5 题，60 秒倒计时 → 点击选项后即时反馈正误 + 解析
4. **结算** — 查看得分、星级、连击、段位升级 → 错题回顾 → 进入下一关
5. **排行榜** — 提交成绩 → 查看 TOP 20 排名
6. **知识手册** — 随时搜索复习全部 40 道急救知识
