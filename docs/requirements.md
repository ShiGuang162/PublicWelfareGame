# 🎮 急救先锋 — 小红花公益急救闯关游戏

## 项目概述
- **项目名称**：急救先锋（小红花急救大闯关）
- **赛题方向**：小红花游戏 — 用游戏传递公益力量（应急急救）
- **目标平台**：Web 网页（H5），跨平台适配 PC 与移动端
- **受众群体**：全年龄段，零基础可上手
- **核心定位**：寓教于乐，通过游戏化答题让用户掌握实用的急救知识，提升全民应急救护能力

---

## 需求背景
应急急救知识是每个人都应掌握的生命技能，但传统科普形式枯燥，用户参与度低。本游戏通过「问答闯关 + 段位进阶 + 小红花收集」的轻竞技模式，将严肃的急救知识转化为轻松有趣的互动体验，激发用户持续学习动力，实现公益价值与游戏乐趣的统一。

---

## 核心功能需求

### 1. 首页（入口页）
- 展示游戏 Logo、标题、公益标语
- 开始游戏按钮（进入关卡选择）
- 我的段位/小红花数量展示
- 急救知识小贴士轮播
- 适配移动端触摸与大屏点击

### 2. 关卡系统（8大急救主题）
| 关卡 | 主题 | 题目数 | 解锁条件 |
|------|------|--------|----------|
| 第1关 | 心肺复苏（CPR） | 5题 | 默认解锁 |
| 第2关 | 止血包扎 | 5题 | 第1关通过 |
| 第3关 | 火灾逃生 | 5题 | 第2关通过 |
| 第4关 | 溺水救援 | 5题 | 第3关通过 |
| 第5关 | 中暑急救 | 5题 | 第4关通过 |
| 第6关 | 异物卡喉（海姆立克） | 5题 | 第5关通过 |
| 第7关 | 地震避险 | 5题 | 第6关通过 |
| 第8关 | 触电急救 | 5题 | 第7关通过 |

- 每关答题倒计时 60 秒（可调整）
- 答对 +20 小红花，答错 -5 秒时间
- 连续答对有连击加分
- 关卡通过后显示知识点总结卡

### 3. 答题系统
- 单选题，每题 4 个选项
- 选中后立即反馈对错（绿色正确/红色错误 + 知识点解析）
- 错误时播放解析动画，强调正确做法
- 进度条展示当前题号/总题数
- 时间条实时倒计时，增加紧张感

### 4. 段位进阶系统（核心竞技机制）
| 段位 | 所需小红花 | 称号 |
|------|------------|------|
| 青铜急救员 | 0-99 | 急救小白 |
| 白银急救员 | 100-299 | 急救学徒 |
| 黄金急救员 | 300-599 | 急救能手 |
| 铂金急救员 | 600-999 | 急救达人 |
| 钻石急救员 | 1000-1499 | 急救专家 |
| 急救大师 | 1500+ | 公益英雄 |

- 段位徽章在首页、结算页、排行榜中展示
- 段位提升时触发升级动画（全屏特效）
- 高段位解锁专属称号和头像框

### 5. 排行榜（竞技性）
- 本地排行榜（按总小红花数排序）
- 展示：排名、昵称、段位、小红花数、通关数
- 支持用户输入昵称提交成绩
- 前三名特殊奖牌标识（金牌/银牌/铜牌）

### 6. 答题结算页
- 本关得分、正确率、用时统计
- 获得小红花数量展示（动画计数）
- 错题回顾（可再次查看错题解析）
- 分享按钮（鼓励传播）
- 下一关/返回首页按钮

### 7. 公益元素融入
- 每关结束后展示「真实案例」或「数据科普」卡片
- 游戏内嵌急救知识速查手册（可单独浏览）
- 通关全部关卡后生成「急救公益证书」
- 所有题目知识点均来自权威急救指南（红十字会、AHA等）

## 非功能需求

### 性能
- 首屏加载 < 2 秒（单页应用，无外部依赖）
- 所有资源本地打包，无网络请求阻塞
- 动画流畅，60fps

### 兼容性
- 支持 Chrome、Safari、Firefox、Edge 最新两版
- 移动端 iOS Safari / Android Chrome 完美适配
- 响应式布局，320px - 1920px 屏幕自适应

### 可访问性
- 字体大小可随系统缩放
- 高对比度配色，色盲友好
- 按钮尺寸 >= 44px（移动端触控标准）

---

## 数据持久化
- 使用 LocalStorage 保存：
  - 用户昵称
  - 当前段位与小红花总数
  - 各关卡解锁状态与最佳成绩
  - 本地排行榜 TOP 20
- 数据结构清晰，支持导出/重置

---

## 题目设计原则
- 每题必须附带「解析说明」，答错时展示教育内容
- 选项设计有干扰性，避免一眼看出答案
- 涵盖判断题场景（如"以下哪种做法是错误的"）
- 结合真实急救流程步骤（如CPR顺序：判断→呼救→按压→吹气）
